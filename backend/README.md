# 后端框架与微服务

放后端框架、微服务相关 Demo：Spring 生态、RPC、服务治理、API 设计等。

**本目录现有内容：**

- **spring-boot/**：Spring Boot 3 多模块（按业务主题：config、elasticsearch、rabbitmq、cassandra、mongodb、multicache、kafka、rsocket、resilience4j、errorhandling 等），在 `spring-boot` 下执行 `mvn compile` 可编译全部。
- **spring-core/**：原 demo-spring-core，Spring Framework 核心多模块（core01～core23：IoC、Bean、XML 配置、生命周期、AOP、JdbcTemplate 等），在 `spring-core` 下执行 `mvn compile` 可编译全部。

可继续添加：Spring Cloud、REST/gRPC 示例、网关/限流/熔断等。
