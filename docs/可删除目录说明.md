根目录 `.gitignore` 已忽略 **target/**，不会提交。若希望清理本地磁盘，可在对应目录执行：

```bash

cd java && mvn clean

# 清理 backend/spring-core
cd backend/spring-core && mvn clean

# 清理 backend/spring-boot 下各模块
cd backend/spring-boot && mvn clean
```

或手动删除各子项目下的 `target` 目录。
