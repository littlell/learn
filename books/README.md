# 读书学习

放技术类书籍的跟读与练习：按书名建子目录，通过源码和示例加深对书中技术原理的理解。

**使用方式：**

- 子目录名 = 书名或书名缩写（如《深入理解 Java 虚拟机》可建 `jvm-3rd` 或 `understanding-jvm`）。
- 每个子目录内可放：书中示例代码、自己敲的 Demo、读书笔记或小结。
- 以「能运行的代码」为主，便于对照书中章节验证原理。

**本目录现有内容：**

- **phoenix-architecture/**：原 `phoenix-architecture-demos`，书籍《凤凰架构》的示例与笔记（如事务隔离实现等），已合并于此。

可放内容示例：随书源码整理、每章对应的小实验、关键知识点的可运行示例。
